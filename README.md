# takhleel - more about products in Uzbek.

An easy tool for knowing more about products without browsing tons of comments.

## Possibilities:
- Sentiment Analysis
- Word Cloud
- Analysis of your products(soon)

## Supports the following:
- Youtube
- Google Maps
- Instagram (soon)
- Online Shops(soon)
- Marketplaces (soon)
